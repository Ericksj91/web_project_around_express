DESCRIPCIÓN: Esta es una API REST desarrollada con Node.js y Express. La API devuelve datos de usuarios y tarjetas que ahora están guardados en una base de datos MongoDB (aroundb), conectada mediante Mongoose. Las rutas están separadas en dos archivos distintos (uno para usuarios y otro para tarjetas) usando express.Router(), para que el código quede más organizado. Se crearon esquemas y modelos de Mongoose para user y card, con validaciones como longitud mínima y máxima de strings, campos obligatorios, y un validador personalizado con expresión regular para los enlaces de avatar e imagen. Los controladores usan métodos de Mongoose como find, findById, findByIdAndUpdate y findByIdAndDelete, junto con el helper orFail() para manejar los casos en que un usuario o tarjeta no existe. También se agregó manejo de errores: si se pide un usuario o tarjeta que no existe, devuelve un 404; si los datos enviados son inválidos, devuelve un 400; y si hay algún problema en el servidor, devuelve un 500. Se implementó además un middleware temporal de autorización que agrega un usuario de prueba a cada petición, utilizado como autor (owner) al crear tarjetas y para dar o quitar likes mediante los operadores $addToSet y $pull. Se usó ESLint con la configuración de airbnb-base para mantener un estilo de código limpio.

TECNOLOGIAS USADAS: Node.js, Express.

Prueba la funcionalidad del proyecto en Postman con los siguientes enlaces: http://localhost:3000/users, http://localhost:3000/cards.
